# FisioControl

Sistema web de gestão clínica desenvolvido com foco em UI/UX e metodologia Scrum.

## Objetivo
Gerenciar pacientes, consultas e acompanhamento clínico.

## Tecnologias/Ferramentas
- Figma
- GitHub Projects
- Scrum

## Estrutura do Projeto
- Wireframes
- Mockups
- Protótipo
- Documentação Scrum

## Funcionalidades
- Cadastro de pacientes
- Agenda
- Histórico clínico
- Controle financeiro
